# MyVitalScan – Documentation légale et information aux participants

## Description du projet

Ce dépôt GitHub regroupe les documents officiels associés au projet de recherche **MyVitalScan**, une application mobile visant à soutenir l’optimisation des soins prénataux grâce à l’utilisation de paramètres biométriques et de l’intelligence artificielle (IA).

Le projet consiste à valider une application capable de mesurer certains paramètres physiologiques à l’aide de la caméra d’un téléphone intelligent. Les mesures obtenues sont comparées à celles d’instruments médicaux validés. Les données recueillies contribuent également à l’entraînement et à l’amélioration d’algorithmes d’IA intégrés à l’application, dans une perspective de fiabilité, d’autonomie et de sécurité.

L’objectif global est de contribuer au développement d’un outil technologique pouvant, à terme, être utilisé dans des contextes cliniques pour améliorer les soins prénataux et la surveillance de la santé.
